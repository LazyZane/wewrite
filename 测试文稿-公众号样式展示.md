### 代码块

#### JavaScript代码
```javascript
// 这是一个JavaScript函数示例
function calculateTotal(items) {
    let total = 0;
    for (let item of items) {
        total += item.price * item.quantity;
    }
    return total;
}

// 使用示例
const shoppingCart = [
    { name: "苹果", price: 5, quantity: 3 },
    { name: "香蕉", price: 3, quantity: 2 }
];

console.log(`总价：${calculateTotal(shoppingCart)}元`);
```
