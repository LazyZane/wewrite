# 1. 风格特点

- 以中文宋体为主要字体
- 排版中规中矩
- 泛黄的报纸底色，非纯黑色的字体
- 不做多余的修饰
- 基本使用默认的风格

```CSS
:root {
	 --article-font-family: "FZBaoSong-Z04S", "SimSun", "Songti SC", "NSimSun", serif;
	 --wewrite-text: rgba(0,  0,  0, 0.8);
	 --wewrite-bg: #f0e6d2;
	 --article-max-width: 900px;
}
```
